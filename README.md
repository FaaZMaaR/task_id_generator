# Тестовое задание: <br>"Создание собственного алфавита идентификаторов"

### Постановка задачи
Есть последовательность идентификаторов, строящаяся по следующим правилам:

1. Первый идентификатор последовательности имеет вид «A1», второй — «A2», третий - «A3» и так далее. За «A9» следует «B1». Следующий после «Z9» имеет вид «A1-A1», потом «A1-A2» и так далее. После «A1-Z9» следует «A2-A1».
2. Максимальная длина идентификатора - десять групп по два символа.
3. В идентификаторах никогда не должны присутствовать буквы «D», «F», «G», «J», «M», «Q», «V» и цифра «0».
4. Нужно на C++ написать библиотечный класс, предназначенный для использования другими программистами, обеспечивающий функцию генерации идентификаторов по этим правилам.

### Реализация
Структура класса IDGenerator:

* Два вспомогательных поля `alphaMap` и `alphaVector` для хранения букв и их индексов с учетом отсутствующих букв;
* Вспомогательная функция `increaseIdentifier`, которая вычисляет следующий одиночный идентификатор из переданного в качестве аргумента. Если получен идентификатор Z9, функция возвращает A1.
* Вспомогательная функция `checkLastIdentifier`, которая проверяет, является ли переданный идентификатор последним.
* Основная функция `generateNextIdentifier`, которая вычисляет следующий идентификатор из переданного в качестве аргумента. Если идентификатор является последним, выбрасывается исключение.
